This project serves as a central hub for my work, highlighting the applications I’ve built, the technologies I work with, and my development journey.

# Features

- Responsive design optimized for desktop and mobile
- Smooth page transition animations
- Modern UI built with Tailwind CSS
- Dedicated pages for projects and about information
- Animated UI elements using Framer Motion
- Optimized images using Next.js Image component
- Scalable component-based architecture

# Getting Started

Clone the repository

```
git clone https://github.com/jcalderaro-96/Career_Portfolio.git
```

Navigate into the project directory

```
cd Career_Portfolio
```

Install dependencies

```
npm install
```

Start the development server

```
npm run dev
```

Open the project in your browser

```
http://localhost:3000
```

# Deployment

This project will be deployed using **Vercel**, the platform built by the creators of Next.js.

Deployment steps will include:

- pushing the repository to GitHub
- importing the project into Vercel
- automatically building and deploying the site
